Inventory Manager

https://github.com/rororowyourkayak/inventory-manager

In this project, I built a web-based inventory management system that allows users to store UPC-A based items.
The user stores a UPC-A code, category, description and quantity for each item they may store.
Additionally, this system allows for photo uploads to be viewed with an item's information.

The system features a user system and input validation based on Laravel.
The application supports importing and exporting of inventory as a CSV or XLSX file.
Project also features an example of recaptcha protection in the contact page.

Frontend

The frontend of this system was built with HTML, CSS, Bootstrap 5, and jQuery.
This project also made use of the DataTables and Select2 jQuery-based libraries.

Backend

The backend of this project was built using Laravel 8, PHP 8 and MySQL.
This system is a CRUD (Create, Read, Update, Delete) application, making database interactions important.

PA Auto Safety Inspection Stations

https://github.com/rororowyourkayak/pa_inspection_stations

This site is live. You can visit at:

https://pa-auto-inspections.com/

In this project, I built a site that displays information about the safety inspection stations in PA.
The purpose of this site is to make a more accessible medium for this information.
This site also features a convenient search tool for users to find stations.

Frontend

The frontend of this system was built with HTML, CSS, Bootstrap 5.
Additionally, the site's search tool was built with the help of Vue.js.

Backend

The backend of this project was built using Laravel 8, PHP 8 and MySQL.

The most important part of this site was extracting information into a database.
This was done by writing a script with Python using the openpyxl library.
After data was extracted, it was then turned into Laravel models using Laravel Excel.
